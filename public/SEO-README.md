## Google Search Console Setup

### 1. Verify Site Ownership
To verify your site with Google Search Console:

1. Go to [Google Search Console](https://search.google.com/search-console)
2. Add your property: `https://wiredliving.blog`
3. Choose HTML tag verification method
4. Copy the verification code
5. Add it to your `.env.local`:
   ```
   NEXT_PUBLIC_GOOGLE_SITE_VERIFICATION=your-verification-code
   ```
6. Redeploy your site

### 2. Submit Sitemap
After verification:
1. Go to Sitemaps section in Search Console
2. Submit: `https://wiredliving.blog/sitemap.xml`

### 3. Request Indexing
For immediate indexing:
1. Use URL Inspection tool in Search Console
2. Enter your URLs
3. Click "Request Indexing"

## Indexing Best Practices

### Content
- Use semantic HTML
- Include alt text for images
- Use proper heading hierarchy (h1, h2, h3)
- Write descriptive meta descriptions

### Performance
- Optimize images (WebP format)
- Minimize JavaScript
- Use CDN for static assets

### Links
- Use descriptive anchor text
- Internal linking between posts
- External links to authoritative sources
